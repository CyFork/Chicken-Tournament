Chicken Tournament:


***VORWORT***: Da dieses Spiel übertrieben sinnlose Gewalt darstellt, empfehle ich das Spiel von Jugendlichen unter 18 Jahren fernzuhalten, auch wenn das Spiel realitätsfern ist. Das liegt im eigenen Ermessen! Ich übernehme keinerlei Verantwortung für (psychische und materielle) Schäden, die durch das Spiel entstehen (insbesondere an Hühnern IRL). Gartenkrallen sind zum Bearbeiten des Gartens da!!



1) Mindestanforderungen:
- 18 Jahre
- 300 MHz
- Microsoft Windows 98
- 32 MB RAM
- 3D Grafikkarte mit DirectX 7 Unterstützung
- DirectX 7.0
- Soundkarte (optional)
- Netzwerkkarte (optional, für Multiplayer)
- DivX 4 für das Intro
- Maus
- Tastatur



2) Installation
Keine Installation erforderlich. Das Spiel läuft auch von CD. Man kann das gesamte Spiel auf die Festplatte kopieren.



3) Spielregeln
Das Spiel selber hat überhaupt keinen Sinn und kein Ziel. Man kann sich als Bauer oder als Huhn in einer 3D Welt bewegen. Als Bauer kann man eine von 4 verschiedenen Waffen einsammeln, die sich in der Welt befinden (Gartenkralle, Golfschläger, Mähdrescher, Plasmakanone). Mit diesen Waffen kann man dann Hühner ausschalten. Bauer gegen Bauer ist nicht möglich. Als Huhn muss man den Bauer mit Eiern bewerfen. Hühner müssen brüten, um Eier zu regenerieren. Währenddessen sind sie verwundbar. Sie haben ausserdem die Möglichkeit zu fliegen. So sind sie schneller und wendiger, und können den Bauer mit Kamikazeangriffen schaden. Allerdings muss man aufpassen, dass man nicht abstürzt. Hühner können nicht unendlich fliegen; sie haben Flugkraft, die im Flug verbraucht wird. Am Boden wird Flugkraft regeneriert. Auch ein Huhn gegen Huhn Kampf ist nicht möglich. 

Als Huhn kann man in einem Hühnerstall Zuflucht suchen. In diesem ist man geschützt (denn der Bauer hat keinen Zugriff) und man kann zudem schneller brüten und Flugkraft regenerieren.

Bauer können Medipacks aufsammeln, die ihm sofort 50 % Energie zurückgeben (bis max. 125 %), Hühner können Eierschachteln aufsammeln, die dem Huhn sofort weitere 16 Eier als Munition bescheren.
Bauer regenerieren ständig aber sehr langsam Energie.

Die Plasmakanone des Bauern hat Munition, wenn sie verbraucht ist, kann man damit nicht mehr schießen. Allerdings regeneriert die Plasmakanone mit der Zeit Munition. Man kann jedoch dann eine andere Plasmakanone einsammeln, die noch Munition besitzt.

Die Frags der Hühner werde in Prozent gerechnet, d.h. pro Ei und verursachten Schaden, unabhängig davon, ob man den Bauer getötet hat oder nicht.

Im Mähdrescher erhält der Bauer nur 1/5 des normalen Schadens, dafür regeneriert er keine Energie.

Man kann im Multiplayermodus oder alleine spielen. Während des Spiels können Bots hinzugefügt oder entfernt werden. Ausserdem kann die KI der Hühner deaktiviert werden (Fun-Mode ;)



4) Steuerung

Tastatur:
- Escape: Menu
- W/A/S/D, Pfeiltasten: Bewegen
- Leer: Springen / Flug starten

- ^ (Taste unter Escape): Konsole öffnen
- Tab: Multiplayer Statistik anzeigen
- H: Hut ändern. Nur Bauer!
- B: Zurücksehen (geht nur als Huhn im Flug, oder als Bauer im Mähdrescher)
- Backspace: Chat-Fenster öffnen
- C: Kamera wechseln

- Print Screen: Screenshot als Bitmap-Datei auf dem Desktop speichern
- Enter: Aus Mähdrescher aussteigen
- F4: Wireframe an/aus (zum Debuggen, kann als Wallhack erachtet werden (unfair?))
- F5: Sound an/aus
- F6: Musik an/aus
- R: Radar an / ausschalten
- Alt+F4: Spiel sofort beenden (na sowas)


Maus (Bauer):
- Sicht rotieren
- Linke Maustaste: Erster Angriff (Stossen, Schlagen, schießen)
- Rechte Maustaste: Sekundärer Angriff (Wegwerfen, hinlegen, aussteigen).

Maus (Huhn):
- Sicht rotieren
- Linke Maustaste: Eier werfen
- Rechte Maustaste: Brüten
- Im Flug: 
  - W/S/Mausrad: Schub regeln
  - A/D: Seitenruder
  - Hoch/Runter: Höhenruder
  - Links/Rechts: Querruder



5.1) Lokale Konfiguration
Im Spiel Escape drücken, und auf Optionen. Die meisten Optionen dürften Selbsterklärend sein. ;) Alle Optionen wirken sich nur lokal aus. Die Auflösung kann während des Spiels geändert werden.
Die Konfiguration wird in der Datei "config.dat" abgespeichert, im selbem Verzeichnis, wie die Exe Datei. Löschen Sie die Datei, um mit Standardwerten zu spielen.

5.2) Serveroptionen
Escape drücken, dann auf Serveroptionen. Diese Optionen betreffen das gesamte Multiplayerverhalten. Hier können Bots, Objekte, etc. gespawnt werden.


6) Multiplayer
Zu Beginn kann man das Protokoll auswählen und dann ein Multiplayerspiel starten oder einem beitreten. Es ist nicht nötig, Singleplayer auszuwählen, da es sich nicht von einem Multiplayerspiel ohne Mitspieler unterscheidet.
Wenn in einem Multiplayerspiel der Server das Spiel verlässt, wird automatisch ein Client neuer Server und das Spiel geht weiter. 
Nur der Serverspieler ist in der Lage, die Serveroptionen zu verändern (über das Menü oder die Konsole).
Das Zusatzprogramm "Register4Lobby.exe" kann benutzt werden, um CT der Liste von DirectPlay Lobbyfähigen Spielen hinzuzufügen. Über eine DirectPlay Lobby kann dann nun automatisch CT gestartet werden. Das funktioniert z.B. beim Microsoft Messenger, der zum Starten von CT benutzt werden kann (nach dem Registrieren steht im Chatfenster dann "CT starten" oder so ähnlich).


8) Ports
Da ich von Natur aus faul bin, benutzt CT DirectPlay für Netzwerk. DirectPlay benutzt die folgenden Ports (für Firewall/Router Probleme):

Primary Connection
  47624     TCP Outbound
Secondary Connection
  2300-2400 UDP Send Receive
  2300-2400 TCP Inbound
  2300-2400 TCP Outbound



9) Konsole
Hier alle Befehle der Konsole (aufrufen mit ^).

- HELP: Alle Befehle anzeigen
- FOG: Nebel an / aus
- INFO: Listet alle Objekte auf
- SPAWN [1-6]: Erstellt ein Objekt in der Welt (1=Huhn, 2=Gartenkralle, 3=Golfschläger, 4=Mähdrescher, 5=Baum, 6=Hühnerstall)
- EXIT: Beendet das Spiel
- KICK [nr]: Entfernt das Objekt mit der ID nr aus dem Spiel
- SPECULAR: Glanzlichter der Objekte aktivieren / deaktivieren (Performance)
- NEWWORLD: Generiert eine neue Landschaft (Objekte bleiben bestehen).
- SAY [Text]: Sendet eine Chat-Nachricht an alle Mitspieler
- PRIV [UID] [Text]: Sendet eine Chat-Nachricht an den Spieler mit der angegebenen UID
- SENSITIVITY: Stellt die Geschwindigkeit der Maus ein (unter 25: schneller)
- GRAVITY: Gravitation ändern
- GORELEVEL: Menge Blut einstellen
- AIRCONTROL: Faktor für Beschleunigung im Sprung
- JUMPSPEED: Sprunggeschwindigkeitsmultiplikationsfaktor relativ zum Gehen
- ALPHA: Alpha blending aktivieren / deaktivieren (Performance)
- NAME: Spielernamen ändern (nur Gross-Buchstaben).
- TYPE [UID]: Spielertyp wechseln (Bauer / Huhn), Server kann auch Typ vom Mitspieler mit der angegebenen UID ändern.
- SITSPEED: Eier pro Sekunde produzieren (fürs 1. Ei).
- THROWTIME: Zeit zwischen zwei Eierwürfen
- REPLENISHPOWER: %/sek Flugkraft regenerieren
- TAKEPOWER: %/sek Flugkraft während Flug verlieren
- DAMAGE: Schaden von Eiern an Bauern (+/- 50%)
- PLASMAREGENERATION: Wiederaufladen der Plasmakanone in Schuss / sek
- PLASMARADIUS: Radius der Plasmaexplosion in Metern
- PLASMARATE: Schussrate der Plasmakanone (in Sekunden pro Schuss)
- KILL [UID]: Selbstmord, oder (nur Server) Spieler mit UID killen
- RESPAWNTIME: Zeit in Sekunden, bis ein toter Spieler wiederbelebt wird.
- HARVESTERSPEED: Multiplikator für die Beschleunigung des Mähdreschers
- HARVESTERROTSPEED: Multiplikator für die Drehgeschwindigkeit des Mähdreschers
- GRAS: Gras aktivieren / deaktivieren


10) Bekannte Bugs
Ich bitte, die folgenden Fehler zu berücksichtigen und zu akzeptieren. Ich bin dafür verantwortlich und suche noch verzweifelt nach Lösungen. Bisher ist es mir nicht gelungen, die Probleme zu entfernen. Ich hoffe, Sie gehören nicht zu den Betroffenen.

- Der Mehrspielermodus läuft ziemlich unstabil. 
- Bei alten Grafikkartentreibern kann das Spiel extrem ruckeln. Auch können Grafikfehler auftreten.
- Das Beitreten in ein Netzwerkspiel kann fehlschlagen. Dabei hängt der Client, und Serverspiel läuft immer langsamer.
- Da das Spiel kein Ziel und keinen Sinn hat, wird es leider schnell langweilig.
